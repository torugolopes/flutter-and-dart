##### .split()
Por meio de um separador na String, retorna uma List
```dart
String.split(pattern)
```

```dart
  var pessoa = 'Victor|Desenvolvedor|31';
  print(pessoa.split('|'));
```
>[Victor, Desenvolvedor, 31]
